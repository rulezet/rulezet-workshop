# rulezet-workshop

Materials for the **Rulezet workshop**, including the slide deck, hands-on exercises, demo material, and supporting resources.

## Contents

This repository is intended to host all the material used for [rulezet](https://rulezet.org/) workshop, such as:

- slide decks
- practical exercises
- sample rules and datasets
- demo scripts and helper material
- notes, references, and workshop-related resources

## Purpose

The goal of this repository is to make the workshop easy to run, reuse, and improve over time.

It can be used by:

- workshop participants
- trainers and facilitators
- contributors to the Rulezet project
- anyone interested in rule management, DFIR, threat hunting, and detection engineering
